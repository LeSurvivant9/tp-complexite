# Reusable LaTeX docs bootstrap

This `docs/` directory is designed to be copied into a new project with minimal edits.

## Template contract

- `main.tex` compiles after loading `metadata.tex`.
- `mystyle.sty` contains no personal identity values.
- `make lint` resolves only project-local paths.

## Edit these files first

- `docs/metadata.tex`
- `docs/sections/*.tex`
- `docs/figures/*`
